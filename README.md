# Smart Financial Health Analyzer

An AI-powered financial analysis tool that generates Big 4 consulting-style insights from company financial data.

## Features
- Input 3 years of company financials
- Auto-computes key ratios: gross margin, net margin, debt-to-equity, ROE
- AI generates EY/PwC-style findings (strengths, risks, watch areas)
- Interactive charts: revenue trends, margins, debt vs equity, cash flow
